Welcome to the Richmond Airdrop

The Richmond Airdrop is a community-run effort focused on helping people get everyday tasks done during a time when many things are changing quickly.

If you are looking for assistance or support, you can request access to the Richmond Airdrop Signal group using the buttons below.

To ensure your request is seen as quickly as possible, please message all three admins below. This helps avoid delays if one admin is unavailable.
